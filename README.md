ToDesk est un classeur numérique pour les étudiants MMI permettant de centraliser les outils et ressources utiles au quotidien.
Contexte du projet
Dans le monde universitaire, les étudiants utilisent de nombreux outils numériques : convertisseurs, correcteurs d’orthographe, plateformes de partage comme Teams, etc. Cependant, la multiplication de ces outils ainsi que la charge de travail importante rendent leur utilisation parfois complexe et peu efficace.
Ce projet vise donc à regrouper plusieurs outils et fonctionnalités dans une seule application afin de faciliter l’organisation et le travail des étudiants en MMI.
Objectifs du projet
Étape 1
Développer une application permettant :
- d’afficher les devoirs à réaliser
- d’ajouter une description détaillée des devoirs
- de joindre des documents ou des PDF
Étape 2 (version actuelle du projet)
Mettre en place un espace personnel pour chaque étudiant, comprenant :
- un dossier individuel
- des casiers pour chaque matière / cours afin d’organiser les fichiers et ressources
Étape 3 (évolution prévue)
Ajouter des fonctionnalités avancées :
- intégration d’une IA via une API pour générer automatiquement :
- des fiches de révision
- des contrôles blancs
- intégration d’un outil de gestion de projet (diagramme de Gantt)
Objectif final
Créer une plateforme centralisée pour les étudiants MMI, facilitant :
- l’organisation des cours
- la gestion des devoirs
- la révision et la collaboration.
