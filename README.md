# ToDesk
Un Classeur MMI pour mes colegues:

Pourquoi le projet :

Dans le monde universitaire, nous sommes amenés à utiliser plusieurs outils informatiques, 
comme les convertisseurs, les correcteurs d’orthographe ou encore des applications de 
partage comme Teams. Mais la multitude de ces outils et la charge de travail trop importante 
à laquelle les étudiants de MMI font face rendent leur travail pénible et difficile. 
Nous avons donc imaginé une application dont la vocation serait de regrouper toutes ces 
exigences afin de rendre le parcours universitaire des étudiants plus facile.

Etape 1:

Dans un premier temps, l’application aurait pour objectif d'afficher tous les devoirs a faire plus la description de ces devoirs qui sera completter par une option d'jouter des docs et pdf.

Etape 2 : (Le projet est ici)

Creer un dossiers pour chaque etudiant avec de casiers pour chaque cours.

Etape 3 :

Liers les casiers avec Chat gpt avec un API, pour que l'IA puisse nous generer de fiches de revisions et de control Blanc.
